Pioche de Masse est une pioche personnalisée permettant de miner plusieurs blocs à la fois.

Par défaut, la pioche permet de miner 1x2 blocs. Le joueur peut ensuite améliorer et configurer la zone de minage jusqu’à 3x3 selon ses préférences.

En s'accroupissant et en effectuant un clic droit, un menu GUI s’ouvre et affiche une grille représentant les blocs qui seront affectés par la pioche. Depuis ce menu, le joueur peut activer, désactiver ou améliorer les blocs minés, moyennant un coût défini dans la configuration.

Tous les éléments sont entièrement configurables :

    messages
    nom de l’objet
    descriptions (lore)
    prix des améliorations
    blocs affectés par la pioche

Command: /givepioche

Ce Skript a été réalisé pour la version 1.21.11 et nécessite les plugins/addons suivants :

    Skript v2.14.2
    Skript-GUI v1.3.2
    SkBee v3.17.1
